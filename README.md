# Headers Formatter

A simple, beautiful tool to convert raw HTTP headers into a clean Python dictionary.  
This is primarily for my personal use â€” but if someone discovers it, then cool! 

Hosted live here: [https://1sujal.github.io/headers-formatter/](https://1sujal.github.io/headers-formatter/)


## Usage

1. Open [the live tool](https://1sujal.github.io/headers-formatter/).
2. Paste your raw headers in the input box.
3. Toggle **Include cookies** if you want them.
4. Watch the formatted Python dictionary appear instantly.
5. Copy output with the **Copy Output** button.
